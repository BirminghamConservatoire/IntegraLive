This directory contains examples of Integra module hosts. libIntegra communicates with each host via host-specific bridge.

Pd/

    Pd/INTEGRA_HOST.pd  -- a host based on pure data, which requires libIntegra with the integra_osc_bridge, which uses an OSC protocol to communicate with Pd using OSC externals provided by mrpeach

    This host requires Pd-extended 0.43 or later
