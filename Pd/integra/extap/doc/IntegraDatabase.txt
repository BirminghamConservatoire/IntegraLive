Communication betwenn the integralive gui and the pd module is done using these:

From Pd to IL:

masterlimiter 	shows active limiters on the 8 dacs
network 	Network connection to slave established
slaveaction 	monitor slave actions
samplesok 	all samples loaded

s1 		sampler playstate
s2 		sampler playstate
s3 		sampler playstate
s4 		sampler playstate

s1vu 		sampler vu meter
s2vu 		sampler vu meter
s3vu 		sampler vu meter
s4vu 		sampler vu meter

s1file 		sampler filename
s2file 		sampler filename
s3file 		sampler filename
s4file 		sampler filename

current 	current cue
next 		next cue

h1out 	harmonizer output vu
h2out 	harmonizer output vu
h3out 	harmonizer output vu

fils2out 	Soprano2 filter output vu

filecdry 	Ecriture filter dry output vu
filecwet 	Ecriture filter wet output vu

revin 	reverb input vu
revout 	reverb output vu

d1in 	delay input vu
d2in 	delay input vu
d3in 	delay input vu
d4in 	delay input vu

d1out 	delay output vu
d2out 	delay output vu
d3out 	delay output vu
d4out 	delay output vu

resin 	resonator in vu meter
resout 	resonator out vu meter
reslim 	resonator limiter indicator

From IL(MIDI input) to Pd:

go		Trigger a cue
stop		stop all cues
plus1		increment next cue number by one
plus10		increment next cue number by ten
minus1		decrement next cue number by one
minus10		decrement next cue number by ten
NOTE: goto0 is only implemented on slave

s1vol		sampler master gain fader
s2vol		sampler master gain fader
s3vol		sampler master gain fader
s4vol		sampler master gain fader

harm		harmonizers master gain fader
rev		reverb master gain fader
reson		resonator master gain fader
delays		delays master gain fader
filts2		filter Soprano2 master gain fader
filtec		filter Ecriture master gain fader
	
transforms 	transformations master gain fader
samplers 	samplers master gain fader
